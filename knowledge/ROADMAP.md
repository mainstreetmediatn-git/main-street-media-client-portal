# Roadmap

Status: Roadmap

Last updated: 2026-07-13

## Phase VI: Business Intelligence Core

Goal:

- establish a richer intelligence layer on top of `BusinessSnapshot`
- add durable business-state history
- improve the operator and customer visibility model

Current status:

- partially scaffolded
- not live as an autonomous runtime

## Phase VII: Agent Orchestrator

Goal:

- coordinate future agents against the canonical snapshot
- route work by stage, tenant, and permissions
- maintain approval and execution traces

Current status:

- roadmap only

## Phase VIII: Operations Operating System

Goal:

- build an operations console for fulfillment, reporting, support, and task routing
- connect manual operations with durable workflow records

Current status:

- roadmap only

## Phase IX: Revenue Intelligence

Goal:

- connect package state, payment events, lead health, and retention signals
- use business data to guide renewals and expansion

Current status:

- roadmap only

## Phase X: Autonomous Growth Operating System

Goal:

- combine the intelligence core, orchestrator, operations console, and revenue signals into a controlled autonomous operating layer

Current status:

- roadmap only

## Rule

Future architectural discussions should be written into this roadmap structure before implementation begins.

## Known Constraints

- Do not claim autonomous runtime behavior until it exists.
- Do not move features into these phases without implementation evidence.
